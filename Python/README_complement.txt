欠損値の補完

[基本機能]
入力ファイル：standardize.pyで処理した結果のcsvファイル
出力ファイル: 入力したファイルと列名同じ、行数が追加されたファイル
　　　　　　　「_rendered」という文字を付加して、同じフォルダに出力します

データ：時間・天気以外は小数点7桁目四捨五入の小数点６桁表示

欠損データを発見した際、条件によって2パターンの動作をします。

１．欠損データの1時間後のデータが存在する場合
　欠損データから1時間後のデータの間を等間隔で補完します
　等間隔で保管されるデータはhumidとtempのみであり、timeとweatherは直前データの値がコピーされます。

２．欠損データの1時間後のデータが存在しない場合
　全ての項目について、直前データの値がコピーされます。

[使用方法]

1.　環境構築

requirements.txt　に従って、パッケージのインストールを行います。

pip install -r requirements.text

2. フォルダ構造の確認

CSV2 : 処理するcsvが入ったフォルダ(standardize.pyの出力フォルダ)
　　　　兼　本プログラムの出力フォルダ

3.　プログラムの実行

python において実行します。

python complement.py

実行ファイルの main.py は同じフォルダ内のとCSV2フォルダを探します。
zipファイルを開いた状態で、動作できる状態になっています。
誤って移動・削除してしまった場合でも、同名フォルダが存在すれば動作します。
