データの欠損値補完

[基本機能]
入力ファイル：CSV_01.csvと同フォーマットのcsvファイル
出力ファイル: CSV_02.csvと同列名のcsvファイル

データ：時間以外は小数点7桁目四捨五入の小数点６桁表示

[使用方法]

1.　環境構築

requirements.txt　に従って、パッケージのインストールを行います。

pip install -r requirements.text

2. フォルダ構造の確認

CSV : 処理したい画像を入れるフォルダ
CSV2 : 処理された画像及びcsvが入るフォルダ

3.　プログラムの実行

python において実行します。

python standardize.py

実行ファイルの main.py は同じフォルダ内のCSVフォルダとCSV2フォルダを探します。
gipファイルを開いた状態で、動作できる状態になっています。
誤って移動・削除してしまった場合でも、同名フォルダが存在すれば動作します。
